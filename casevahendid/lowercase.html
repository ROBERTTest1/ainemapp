CASE vahendite konspekt

Lowercase vahendid

  • "Alumise taseme" CASE vahendid on teostusele ja mudelist päris tarkvaratooteni saamiseks vajalikud programmid, mis toetavad erinevaid faase toote valmimisel. Näiteks andmebaasi struktuuri ja koodi genereerimist, erinevaid haldus ja testimis võimalusi.

  • Lowercase vahendid mida olen siiani õppimisel kasutanud on peamiselt Visual Studio Code.

    • Veel pole kasutanud järgmisi programme:

    • MySQL -

      • MySQL on avatud lähtekoodiga relatsiooniline andmebaasihaldussüsteem (RDBMS), mida kasutatakse andmete struktureeritud kujul salvestamiseks ja haldamiseks. Seda kasutatakse laialdaselt veebirakendustes ja suuremates süsteemides.

    • Maven -

      Maven on populaarne avatud lähtekoodiga projektihalduse ja ehitamise tööriist, mida kasutatakse peamiselt Java projektides. Maven automatiseerib projektide koostamist, sõltuvuste haldamist ja projektistruktuuri standardiseerimist. See põhineb POM-il (Project Object Model), mis kirjeldab projekti metaandmeid ja konfiguratsiooni.
